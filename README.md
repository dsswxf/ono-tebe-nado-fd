# Учебный проект Яндекс Практикума

Проект выполнен в рамках обучения на Яндекс Практикуме.  
Задача — повторить предложенный макет и сверстать страницу.

## Реализовано
- HTML-разметка страницы
- CSS-стилизация по макету
- Использование шрифтов и изображений

## Технологии
- HTML5
- CSS3
